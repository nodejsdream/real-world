# real-world

Sample from `redux-saga`


## Install

```sh
$ npm install
```
or

```sh
$ yarn
```

## Usage

Start api server
```sh
$ npm run api
```
Start project  server
```sh
$ npm start
```

Start all at once
```sh
$ npm run all
```
